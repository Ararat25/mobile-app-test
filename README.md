# Mobile App Test

## Описание проекта

Этот проект представляет собой мобильное приложение, разработанное с использованием Flutter.

## Архитектура

Проект состоит из следующих основных компонентов:
- android
- ios
- lib
- linux
- macos
- web
- windows

Каждый компонент содержит свои файлы и настройки.

## Основные файлы

- analysis_options.yaml
- build.gradle.kts
- pubspec.yaml
- test/widget_test.dart

## Дополнительная информация

Для более подробной информации о каждом компоненте, пожалуйста, обратитесь к соответствующим файлам в репозитории.
