# Mobile App Test

## Структура проекта

Проект состоит из следующих директорий:
- android
- ios
- lib
- linux
- macos
- pubspec.lock
- pubspec.yaml
- test
- web
- windows

### Описание компонентов

- **android**: содержит файлы для сборки приложения под Android.
- **ios**: содержит файлы для сборки приложения под iOS.
- **lib**: содержит основной файл приложения (main.dart).
- **linux**: содержит файлы для сборки приложения под Linux.
- **macos**: содержит файлы для сборки приложения под macOS.
- **pubspec.lock**: файл блокировки зависимостей.
- **pubspec.yaml**: файл с зависимостями и настройками проекта.
- **test**: содержит файлы для тестирования приложения.
- **web**: содержит файлы для веб-версии приложения.
- **windows**: содержит файлы для сборки приложения под Windows.
